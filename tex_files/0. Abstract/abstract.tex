\begin{abstract}
Deployed literature-search engines force a choice between two retrieval paradigms: structured retrieval, which matches exact terms and typed metadata and so answers a query deterministically and auditably, and semantic retrieval, which ranks papers by meaning and so surfaces relevant work whose wording differs from the query. This thesis designs, builds, and evaluates a scholarly search system that combines the two, built to production standard for Latent Knowledge, an industry partner whose users are biomedical researchers. A typed knowledge graph of approximately 150 million biomedical works from OpenAlex anchors a three-stage cascade: structured graph retrieval returns every work satisfying the query, dense retrieval ranks that set by meaning, and a cross-encoder re-ranks the final results. A language model performs the two functions the structured stage requires, building the keyword index offline and translating each query into an intermediate representation, while a deterministic compiler constructs the database query, preserving the correctness and auditability that systematic search demands.
Benchmarks for each module select the model deployed for that module, and together they establish that each layer of the design can be built to a high standard. The deployed translation model, GPT-5.4-nano, correctly structures 93.7 percent of a hand-built query benchmark; the same model produces the strongest overall keyword index, leading two of the three keyphrase benchmarks; and recent general-purpose models equip both ranking stages, outperforming their scientific fine-tuned counterparts. Storing the embeddings as 8-bit integers halves the memory the index requires at no measurable cost, which is what lets the full graph reside in memory. The partner receives a production-ready system in which every model choice rests on measured evidence and remains substitutable as stronger models emerge. How the assembled cascade compares against the engines researchers already use, Google Scholar, PubMed, and Semantic Scholar, is reported in the final submission.
\end{abstract}
